## Deploy Vps

- `git clone https://github.com/BukanDev/spill-and-drink`
- `cd spill-and-drink`
- `pip3 install -U -r requirements.txt`
- `nano sample.env`
- `cp sample.env .env`
- `screen -S spill`
- `python3 kenkan.py`


## Command

```
/spill - spill dulu
/drink - minum dulu
/donasi - donasi ke owner bot
/request - request spill bikinan mu
```
